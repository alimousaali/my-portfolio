## 🌟 About the Project
This is a modern, responsive, and high-performance single-page portfolio designed to highlight technical expertise in Data Science and Analytics. It features a sleek dark-themed UI with interactive elements to provide an engaging user experience.

## 🛠️ Tech Stack
- **HTML5**: Semantic structure for better SEO and accessibility.
- **CSS3**: Custom properties (variables), Flexbox, and Grid for a modern layout.
- **JavaScript (Vanilla)**: For interactive components like the sticky navbar, smooth scrolling, and the certificates slider.
- **Google Fonts**: Utilizing 'Inter' and 'Space Grotesk' for professional typography.
- **Font Awesome**: For high-quality vector icons.

## ✨ Key Features
- **Modern UI/UX**: Dark mode aesthetic with "Glassmorphism" effects and animated background orbs.
- **Interactive Certificates Slider**: A custom-built slider to showcase professional certifications.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.
- **Project Showcase**: Organized display of data science projects with direct links to repositories.
- **Contact Integration**: Easy-to-use contact section with social media links.
